Powerful Features for Analytics Debugging

Everything you need to debug, monitor, and analyze Firebase Analytics events in real-time

Real-time Monitoring

Connect to Android devices via ADB and monitor Firebase Analytics events as they happen. See events flow in real-time with live capture functionality.

Event Visualization

Beautiful tree and JSON views for event parameters. Easily navigate complex nested data structures with expandable sections and highlighting.

Error Detection

Automatically detect and highlight Firebase Analytics errors with detailed error codes, messages, and original values for easy debugging.

Advanced Filtering

Filter events by name, origin, error status, parameters, and more. Search within event parameters to find exactly what you're looking for.

Session Archives

Automatically archive debugging sessions with timestamps and metadata. Load previous sessions for analysis and comparison.

Export & Integration

Export events to JSON, send to webhooks for automated analysis, or upload to Google Sheets for further processing and reporting.
